# raw

Immutable source documents for the wiki pipeline.

## Rules
- Add new sources here (markdown, PDFs, notes, exports, transcripts).
- Do not rewrite source meaning during ingestion.
- If cleanup is required, preserve the original and create a normalized copy.

## Suggested naming
- `YYYY-MM-DD-topic-source.md`
- `YYYY-MM-DD-topic-source.pdf`
